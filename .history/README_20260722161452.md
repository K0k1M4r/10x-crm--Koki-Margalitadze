# 10x CRM

## About

10x CRM is a browser-based Customer Relationship Management (CRM) application built with HTML, CSS, and JavaScript.

The application helps users manage clients, track sales opportunities, organize customer information, and monitor business activity through a clean dashboard interface.

This project was created as part of a front-end web development assignment, focusing on building a complete application using vanilla JavaScript, modular code organization, and browser-based data storage without relying on external frameworks.

---

## Features

## Authentication

- User registration
- User login
- Session management using LocalStorage
- Protected pages with authentication guard
- Logout functionality
- Form validation with multiple error messages

---

## Dashboard

- Personalized welcome message
- Live clock
- Total clients statistic
- Active deals statistic
- Won revenue calculation
- New clients this week statistic
- Sales pipeline overview
- Recent clients display

---

## Client Management

- Load client data from DummyJSON API
- Store client data locally using LocalStorage
- Add new clients
- Delete clients
- Update client status
- Search clients by name or company
- Filter clients by status:
  - Lead
  - Contacted
  - Won
  - Lost
- Sort clients by:
  - Newest
  - Name A-Z
  - Deal value
- Client detail modal
- Add notes to clients
- Reminder notification feature

---

## Profile Management

- View user profile information
- Edit full name and company
- Change password
- Reset CRM client data

---

## User Experience

- Light and dark theme toggle
- Toast notifications
- Input validation
- Error handling for API requests
- Responsive layout
- Clean dashboard interface

# How to Run

## Option 1 — VS Code Live Server

1. Clone this repository.

2. Open the project folder in Visual Studio Code.

3. Install the Live Server extension.

4. Right-click `index.html`.

5. Select:

```
Open with Live Server
```

6. The application will open in your browser.

---

## Option 2 — Local Development Server

Open the project folder using any local development server and launch:

```
index.html
```
